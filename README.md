# GreenLight Developed in Rust

## Toolchain Installed

```commandline
rustup update

rustup toolchain list
```

## Project Installation

```commandline
rustc --version

cargo --version
```

## Faster Linking

### Installing zld

```commandline
brew install michaeleisel/zld/zld
```

### Create the .cargo/config.toml file

```commandline
mkdir .cargo/
touch .cargo/config.toml
```
